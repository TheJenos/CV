Pilz.IO

(Web Application)

Pilz.IO is a custom-built e-commerce web application for an online pharmacy that runs on the J2EE platform, using JSP & Servlet, MySQL, Angular JS, JavaScript, CSS, HTML, Hibernate, and Bootstrap. The application is hosted on a Glassfish server and uses several custom frameworks developed in-house, including a JSRMI Server, a basic neural network framework, a JavaEE Render, and an Input Guard .
